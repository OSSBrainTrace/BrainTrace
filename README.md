# BrainTrace

지식 그래프 기반 질의응답 시스템

## 📋 프로젝트 정보

- **버전**: 2.0.0
- **라이선스**: MIT License
- **저작권**: © 2025 brainTrace

## 🚀 기술 스택

### Backend
- FastAPI
- Neo4j (그래프 데이터베이스)
- SQLite
- OpenAI API
- LangChain
- PyTorch

### Frontend
- React 19
- Electron
- D3.js (그래프 시각화)
- Vite
